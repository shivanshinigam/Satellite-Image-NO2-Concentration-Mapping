Project Summary

Title: AeroSpectra

Objective: The project aims to develop an AI/ML-powered platform that generates fine spatial resolution air quality maps, specifically focusing on NO₂ (Nitrogen Dioxide) concentrations. The platform will utilize satellite data, process it to account for cloudy conditions, and validate results using independent data sources.

Key Features:

  • Data Processing: Integrates satellite data with machine learning algorithms to create detailed NO₂ concentration maps.
  • Cloud Handling: Implements techniques to estimate NO₂ levels accurately even under cloudy conditions.
  • User Interaction: Allows users to search for locations, view real-time NO₂ concentrations, and interact with detailed visualizations on a map.
  • Real-Time Data Integration: Provides live updates of NO₂ concentrations, enhancing situational awareness.
  • Visualization: Displays the processed data in an intuitive, user-friendly manner, using interactive maps and charts.

Target Users: Researchers, government bodies, and environmental agencies who require precise air quality data for analysis and decision-making.

Technologies Used:

  • Front-End: HTML, CSS, JavaScript (React.js)
  • Back-End: Python (Flask), Machine Learning Libraries (e.g., Scikit-learn, TensorFlow)
  • Mapping: Leaflet.js for interactive maps, Folium for Python-based map generation.
  • Data Sources: Satellite data, real-time APIs for live NO₂ concentration.
